# Документация к API

Базовый путь: `/api/v1`

---

## API для сотрудников авиакомпании

### GET

| Endpoint | Описание |
|----------|----------|
| `/api/v1/airlines/flights` | Список рейсов |
| `/api/v1/airlines/tickets` | Список талонов |
| `/api/v1/airlines/flights/{id}` | Получить рейс по id |
| `/api/v1/airlines/tickets/{ticketNumber}` | Получить талон по номеру билета |
| `/api/v1/airlines/flights/{id}/boardingPasses` | Все талоны на посадку для конкретного рейса |
| `/api/v1/airlines/reports` | Отчётность об основных показателях системы (конкретные метрики определим позже) |

### POST

| Endpoint                                             | Описание | Тело запроса |
|------------------------------------------------------|----------|--------------|
| `/api/v1/airlines/flights/{id}/payment/processTopUp` | Начислить на каждый посадочный талон рейса сумму | `amount` — денежная сумма |

### PUT

| Endpoint | Описание | Параметры запроса |
|----------|----------|------------------|
| `/api/v1/airlines/validatePoint` | Одобрить заявку кафе на подключение к системе по id кафе | `pointId` — id кафе |

### DELETE

| Endpoint | Описание |
|----------|----------|
| `/api/v1/airlines/flights/{id}/payment/restore` | Откатить все начисления на текущий рейс |
| `/api/v1/airlines/ticket/{ticketNumber}/payment/restore` | Откатить начисление на конкретный посадочный талон |

## API для сотрудников кафе

### GET

| Endpoint | Описание | Параметры запроса |
|----------|----------|------------------|
| `/api/v1/points/checkBalance` | Проверить баланс по номеру билета | `ticketNumber` — номер посадочного талона |

### POST

| Endpoint | Описание | Тело запроса |
|----------|----------|--------------|
| `/api/v1/points/pay` | Списать сумму с посадочного талона | `amount` — сумма списания, `ticketNumber` — номер посадочного талона |
